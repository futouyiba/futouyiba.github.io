这个博客的目标和宗旨

Apr 18, 2017阅读次数 : -

世界已经被公众号占领，已经没人看博客了，为什么还要做博客呢？

公众号是一个深度绑定微信的产品。而我们不但渴望连接，还渴望更广阔的连接。因此我们做了这个博客。

那好，不用公众号，那么也可以架设在新浪、网易等博客平台上呀？

各个博客平台在我们写博客时都非常方便，但看帖时经常需要有其账号登录才能阅读全文、交流互动，而【每个人在每个平台上都有一个账号需要管理】本身就是巨大的浪费，和互联网精神十分违背。我们看到国外大部分的网站都不会这样的强人所难。我们所希望的方式是：

  • 如果您是第一次来到这个博客，只需要做一些简单的bot验证就能与我们互动
  • 如果您想长期互动，又有google、微信等账号，直接用这些第三方账号就可登录
  • 如果恰好您没有其它账户，我们也将提供注册用户的功能
    当然，现在这些还没有进行开发。不出意外的话开发会在数月后完成。

此博客会是什么语言的？

坦率说还没想好。如果有时间的话我们希望能用中文和英文把自己的想法向全世界展示，并和所有感兴趣的人交流。

此博客是关于什么的？

我想会包含我们所珍视的一切：

  • 游戏开发的心得与思路
  • 其它领域开发的学习分享
  • 玩游戏的有趣的事情与打法经验
  • 对人生、社会的思考
  • 其它八竿子打不着的东西

See ya！
